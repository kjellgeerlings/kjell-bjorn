CSS Tricks

CSS-tricks is een website waar ervaren webdesigners hun kennis geven over een bepaald onderwerp binnenin een programmeertaal (voornamelijk CSS). Wij hebben veel naar hun advies gekeken. De beschrijvingen geven een dieper inzicht in wat de code betekent en doet, dat helpt ons HBO-ers meer kennis op te doen dan als we alleen de code kopieerden.

Stack Overflow

De website van Stack Overflow is een gigantisch forum bord waar mensen hun vragen kunnen stellen over een idee of lastige code en dan antwoord krijgen van andere programmeurs. De site helpt met antwoord krijgen op vrij specifieke vragen die wij ons afvragen.

Over Web Design

Kjell en Bjorn hebben samen een site in elkaar gezet en de gevraagde opdrachten voltooid. We hebben allebei de opdrachten zelf gemaakt en zijn bij elkaar gekomen om de site in elkaar te zetten met de beste aspecten van ons beide. Het vak van Webdesign 1 legt een focus op het gebruik van CSS en HTML, de eerste 3 weken gingen voornamelijk om het gebruik van HTML en de basis daarvan gebruiken. Na de basis leggen kregen wij bijna oneindige vrijheid door middel van CSS. Creativiteit en vrijheid samengebracht hebben ons de kans gegeven om een indrukwekkende eerstejaars site te bouwen.
